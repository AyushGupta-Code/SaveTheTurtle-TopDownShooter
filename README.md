# Save the TURTLE

Created a **Top-Down Shooter Game** using **Godot Engine**, implementing **A\*** pathfinding and **FSM/BTrees** with enemy NPC AI logic.

---

## To Run the Project

1. **Install** Godot Engine **v4.5**  
2. **Clone** the repo and open the project from Godot (`project.godot`)  
3. **Run** the project from **`world_scene`**
